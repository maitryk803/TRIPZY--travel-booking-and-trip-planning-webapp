## 🌍 TRIPZY – Travel Booking & Trip Planning Web App

TRIPZY is a beginner-friendly travel booking and trip planning web application built using HTML, CSS, and JavaScript. The project is designed to help users explore destinations, discover hotels, plan trips, estimate travel budgets, and manage favorite locations through an interactive and user-friendly interface.

## ✨ Features

- 🔍 Search destinations worldwide
- 📅 Select check-in date and travel details
- 👥 Choose number of travelers
- 🌎 Explore popular travel destinations
- 🏨 Browse top-rated hotels
- 🗺️ Trip planning tools
- 💰 Budget estimation feature
- ❤️ Save and manage favorite destinations
- 📱 Responsive and modern user interface

## 🏠 Home Page

The homepage provides a travel search experience where users can:

- Enter a destination (e.g., Paris, Tokyo, Bali)
- Select check-in date
- Choose the number of travelers
- Search and explore travel options
